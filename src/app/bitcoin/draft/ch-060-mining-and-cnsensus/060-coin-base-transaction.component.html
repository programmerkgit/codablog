コインベーストランザクション

マイニングによってビットコインが発行せれますが、この発行はトランザクションとして表現されます。

このトランザクションはUTXOを持たない特別なトランザクションで、コインベーストランザクションと呼びます。

マイナーは候補ブロックを構築するとき、最初のトランザクションにこのコインベーストランザクションを含めます。

コインベーストランザクションのアウトプットとして、マイナーは自分のアドレス宛に12btcとトランザクション手数料の合計金額を入力します。

マイニングに成功すると、マイナーは報酬としてコインベーストランザクションのアウトプットを受け取ることになります。

Coin base transaction

First transaction included in candidate block is coinbase transaction

Coinbase transaction is a special transaction minner added for the reward of minning

Coinbase transaction has an output value of 12.5btc(issue of btc) + sum of fees of transactions

Coinbase transaction has no utxo, and generate btc.

The recipient address of coinbase of transaction is minner

If minner succeed in minning, minner get those rewards as the fee of validation of blockchain
